Sobre o Bootcamp Front-end

Bem-vindo ao meu portfólio do Bootcamp de Front-end! Este Bootcamp é um programa intensivo que cobre os principais conceitos e ferramentas para o desenvolvimento web moderno. O curso é dividido em três módulos principais: Fundamentos de Front-end Web, Web API e React.

Durante o Bootcamp, eu tive a oportunidade de aprender e praticar HTML, CSS, JavaScript/TypeScript, e o framework React. Cada módulo é composto por aulas teóricas, exercícios práticos e projetos que demonstram a aplicação dos conceitos aprendidos.

Estrutura do Curso

Módulo 1 - Fundamentos de Front-end Web

Este módulo cobre os conceitos básicos de desenvolvimento web, incluindo HTML, CSS, e JavaScript. Nele, aprendi a criar páginas web estáticas e interativas, além de estilizar elementos para proporcionar uma melhor experiência ao usuário.

Módulo 2 - Web API

No módulo de Web API, aprendi sobre como criar e consumir APIs, além de entender como funciona o protocolo HTTP, manipulação de JSON e técnicas de segurança como autenticação e autorização.

Módulo 3 - React

O módulo de React introduziu-me ao popular framework JavaScript para criação de interfaces de usuário. Aprendi a criar componentes reutilizáveis, gerenciar estados e trabalhar com hooks, além de explorar o Next.js para renderização do lado do servidor e testes automatizados.

Por que este Bootcamp?

Escolhi este Bootcamp porque ele oferece uma formação completa em desenvolvimento Front-end, cobrindo desde os fundamentos até ferramentas e técnicas avançadas. Ao final do curso, estarei pronta para desenvolver aplicações web modernas, usando as melhores práticas do mercado.
