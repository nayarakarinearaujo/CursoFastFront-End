O que é CSS Grid Layout?

O CSS Grid Layout é uma técnica de layout que permite criar layouts complexos e responsivos com facilidade. Ele fornece uma maneira eficiente de criar e organizar layouts de duas dimensões (linhas e colunas) usando uma sintaxe simples e poderosa.

Conceitos Básicos do CSS Grid

Contêiner Grid:

O contêiner grid é o elemento pai que define a área onde o grid será aplicado. Para criar um grid, você deve definir o contêiner com a propriedade display: grid ou display: inline-grid.

.grid-container {
    display: grid;
}

Linhas e Colunas:

O grid é dividido em linhas e colunas, e você pode definir seu layout usando as propriedades grid-template-rows e grid-template-columns. Essas propriedades definem a estrutura do grid.

.grid-container {
    display: grid;
    grid-template-columns: 1fr 2fr 1fr; /* 3 colunas com diferentes larguras */
    grid-template-rows: 100px 200px; /* 2 linhas com diferentes alturas */
}

Áreas do Grid:

Você pode nomear áreas específicas do grid usando a propriedade grid-template-areas. Isso facilita a colocação de itens em locais específicos dentro do grid.

.grid-container {
    display: grid;
    grid-template-areas:
        "header header header"
        "sidebar main main"
        "footer footer footer";
}

Itens do Grid:

Os itens dentro do grid são os elementos filhos do contêiner grid. Você pode posicionar esses itens em linhas e colunas específicas usando as propriedades grid-row e grid-column, ou utilizando áreas nomeadas.

.header {
    grid-area: header;
}
.sidebar {
    grid-area: sidebar;
}
.main {
    grid-area: main;
}
.footer {
    grid-area: footer;
}

Propriedades Importantes do Grid

grid-template-columns e grid-template-rows:

Define o número e o tamanho das colunas e linhas do grid.

grid-template-columns: 1fr 2fr; /* 1 parte e 2 partes de largura */
grid-template-rows: auto 100px; /* Altura automática para a primeira linha e 100px para a segunda linha */

grid-area:

Define uma área específica do grid onde o item deve ser colocado.

.item {
    grid-area: 1 / 2 / 3 / 4; /* Linha de início / Coluna de início / Linha de fim / Coluna de fim */
}

grid-gap:

Define o espaçamento entre as células do grid.

grid-gap: 10px; /* 10px de espaçamento entre linhas e colunas */

justify-items e align-items:

Controlam o alinhamento dos itens dentro de suas células.

justify-items: center; /* Alinha os itens horizontalmente no centro das células */
align-items: start; /* Alinha os itens verticalmente no início das células */

justify-content e align-content:

Controlam o alinhamento do grid dentro do contêiner.

justify-content: center; /* Alinha o grid horizontalmente no centro do contêiner */
align-content: space-between; /* Distribui o grid verticalmente com espaço igual entre as linhas */

Exemplos Práticos

Layout Básico:

Um layout com três colunas e duas linhas.

<div class="grid-container">
    <div class="item">1</div>
    <div class="item">2</div>
    <div class="item">3</div>
    <div class="item">4</div>
    <div class="item">5</div>
    <div class="item">6</div>
</div>

.grid-container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    grid-template-rows: repeat(2, 100px);
    gap: 10px;
}

Layout Responsivo:

Um layout que ajusta o número de colunas com base no tamanho da tela.

.grid-container {
    display: grid;
    grid-template-columns: repeat(auto-fill, minmax(100px, 1fr));
    gap: 10px;
}

Layout com Áreas Nomeadas:

Um layout com áreas nomeadas para header, sidebar, main e footer.

.grid-container {
    display: grid;
    grid-template-areas:
        "header header header"
        "sidebar main main"
        "footer footer footer";
    grid-template-columns: 1fr 3fr;
    grid-template-rows: 60px 1fr 40px;
}

Conclusão

O CSS Grid Layout é uma ferramenta poderosa para criar layouts complexos e flexíveis com uma abordagem mais intuitiva e menos dependente de hacks antigos. Ele permite que você crie designs responsivos e adaptáveis com facilidade, simplificando a estruturação e o alinhamento dos elementos da página.
