Variáveis CSS

No CSS, as variáveis são um recurso poderoso que permite reutilizar valores de propriedades em diferentes partes do seu código, facilitando a manutenção e a consistência do design. Elas são chamadas de CSS Custom Properties e são suportadas pela maioria dos navegadores modernos.

Como Funciona

As variáveis CSS são definidas dentro de um seletor e podem ser usadas em qualquer lugar dentro desse seletor ou em seus descendentes. As variáveis são declaradas com dois hífens (--) seguidos pelo nome da variável, e seu valor é atribuído usando a sintaxe --nome-da-variavel: valor;.

Definição e Uso

Definição de Variável

Para definir uma variável, você a declara dentro de um seletor. Normalmente, isso é feito no seletor :root, que é o seletor mais alto e permite que a variável seja acessada em qualquer lugar do CSS.

:root {
    --primary-color: #3498db;
    --font-size: 16px;
    --spacing: 10px;
}

Uso de Variável

Para usar uma variável, você a referencia com a função var().

body {
    color: var(--primary-color);
    font-size: var(--font-size);
    margin: var(--spacing);
}

Exemplo Completo

Aqui está um exemplo completo de como você pode usar variáveis CSS para criar um layout consistente e fácil de manter:

<!DOCTYPE html>
<html lang="pt-BR">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemplo de Variáveis CSS</title>
    <style>
        :root {
            --primary-color: #3498db;
            --secondary-color: #2ecc71;
            --background-color: #f4f4f4;
            --font-size: 16px;
            --padding: 20px;
            --border-radius: 5px;
        }

        body {
            background-color: var(--background-color);
            font-size: var(--font-size);
            margin: 0;
            padding: 0;
        }

        .container {
            padding: var(--padding);
            border-radius: var(--border-radius);
            background-color: var(--primary-color);
            color: #fff;
            max-width: 800px;
            margin: 0 auto;
        }

        .button {
            display: inline-block;
            padding: var(--padding);
            border-radius: var(--border-radius);
            background-color: var(--secondary-color);
            color: #fff;
            text-decoration: none;
            text-align: center;
            transition: background-color 0.3s;
        }

        .button:hover {
            background-color: #27ae60;
        }
    </style>
</head>
<body>
    <div class="container">
        <h1>Bem-vindo ao Exemplo de Variáveis CSS</h1>
        <a href="#" class="button">Clique Aqui</a>
    </div>
</body>
</html>

Vantagens das Variáveis CSS

  • Consistência: Garantem que os mesmos valores sejam usados em todo o site, facilitando a manutenção e atualizações.
  • Facilidade de Manutenção: Alterar o valor de uma variável atualiza todos os lugares onde ela é usada.
  • Escopo: As variáveis podem ter escopo global (no :root) ou local (em um seletor específico), oferecendo flexibilidade.
  • Reutilização: Facilita a reutilização de valores, como cores e tamanhos, sem repetir o mesmo código.

Considerações

  • As variáveis CSS são dinâmicas, o que significa que você pode alterá-las com JavaScript e afetar a aparência do site em tempo real.
  • Elas não são compatíveis com versões muito antigas de navegadores, então, se a compatibilidade com navegadores antigos for importante, você pode precisar usar um polyfill ou considerar outras soluções.

As variáveis CSS são uma adição moderna e poderosa ao CSS que simplifica a estilização e a manutenção de seus projetos.
