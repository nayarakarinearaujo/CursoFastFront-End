Funções CSS

No CSS, funções são uma maneira de aplicar cálculos e transformações dinâmicas aos valores das propriedades. Elas oferecem uma forma de manipular e calcular valores diretamente no CSS, proporcionando flexibilidade e precisão na criação de layouts e estilos.

1. calc()

A função calc() permite realizar cálculos diretamente no CSS. Você pode usar calc() para definir valores dinâmicos, como larguras e alturas, baseados em cálculos matemáticos.

property: calc(expression);

Exemplo:


.container {
    width: calc(100% - 20px); /* Largura total menos 20 pixels */
    margin: calc(1rem + 5px); /* Margem calculada a partir de rem e pixels */
}

2. min()

A função min() retorna o menor valor entre os valores fornecidos. É útil para garantir que um valor não exceda um determinado limite.

property: min(value1, value2);

Exemplo:


.box {
    width: min(80%, 400px); /* A largura será 80% da largura do contêiner ou 400 pixels, o menor dos dois */
}

3. max()

A função max() retorna o maior valor entre os valores fornecidos. Isso garante que um valor tenha pelo menos um determinado limite.

property: max(value1, value2);

Exemplo:


.box {
    width: max(200px, 50%); /* A largura será 200 pixels ou 50% da largura do contêiner, o maior dos dois */
}

4. clamp()

A função clamp() é usada para definir um valor que está dentro de um intervalo de valores. É útil para definir tamanhos de fonte responsivos, por exemplo.

property: clamp(min, preferred, max);

Exemplo:


.text {
    font-size: clamp(16px, 4vw, 24px); /* O tamanho da fonte será pelo menos 16 pixels, idealmente 4% da largura da tela, e no máximo 24 pixels */
}

5. var()

A função var() é usada para acessar o valor de uma variável CSS definida anteriormente. As variáveis são úteis para garantir consistência e facilitar a manutenção do código.

property: var(--variable-name);

Exemplo:


:root {
    --main-color: #3498db;
}

.button {
    background-color: var(--main-color); /* Usa a variável definida para a cor de fundo */
}

Conclusão

As funções CSS, como calc(), min(), max(), clamp(), e var(), oferecem uma poderosa ferramenta para criar layouts flexíveis e responsivos. Elas permitem que você faça cálculos e ajustes dinâmicos diretamente no CSS, melhorando a eficiência e a consistência do design.
