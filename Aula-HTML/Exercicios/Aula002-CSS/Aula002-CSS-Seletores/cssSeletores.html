Explicação de Seletores CSS

1. Seletor de ID

Sintaxe: #id

Descrição: Seleciona um elemento específico com um id único. IDs devem ser exclusivos em uma página, ou seja, cada id deve ser usado apenas uma vez.

<div id="header">Cabeçalho</div>
#header {
    background-color: gray;
    color: white;
}
Cabeçalho

2. Seletor de Classe

Sintaxe: .classe

Descrição: Seleciona todos os elementos com uma classe específica. Múltiplos elementos podem compartilhar a mesma classe.

<p class="highlight">Texto destacado</p>
<p class="highlight">Outro texto destacado</p>
.highlight {
    background-color: yellow;
    font-weight: bold;
}

Texto destacado

Outro texto destacado

3. Seletor de Atributo

Seletor de Atributo Presente:

Sintaxe: [atributo]

Descrição: Seleciona elementos que têm um atributo específico, independentemente do valor desse atributo.

<input type="text" placeholder="Nome">
<input type="password" placeholder="Senha">
[type] {
    border: 1px solid black;
}

Seletor de Atributo Igual:

Sintaxe: [atributo="valor"]

Descrição: Seleciona elementos cujo atributo tem um valor exato.

<input type="text" placeholder="Nome">
<input type="password" placeholder="Senha">
[type="text"] {
    background-color: lightblue;
}

Seletor de Atributo Começa com:

Sintaxe: [atributo^="valor"]

Descrição: Seleciona elementos cujo atributo começa com um valor específico.

<a href="https://example.com">Link 1</a>
<a href="http://example.com">Link 2</a>
[href^="https"] {
    color: green;
}
Link Seguro Link Não Seguro

Seletor de Atributo Contém:

Sintaxe: [atributo*="valor"]

Descrição: Seleciona elementos cujo atributo contém um valor específico.

<div class="box-primary">Caixa Primária</div>
<div class="box-secondary">Caixa Secundária</div>
[class*="box"] {
    padding: 10px;
}
Caixa Primária
Caixa Secundária

4. Seletor de Estado

:hover:

Sintaxe: elemento:hover

Descrição: Aplica estilos quando o usuário passa o mouse sobre um elemento.

<a href="#">Link</a>
a:hover {
    color: red;
}
Passe o mouse aqui

:focus:

Sintaxe: elemento:focus

Descrição: Aplica estilos ao elemento que tem o foco (por exemplo, um campo de formulário que está sendo editado).

<input type="text" placeholder="Digite algo">
input:focus {
    border: 2px solid blue;
}

:active:

Sintaxe: elemento:active

Descrição: Aplica estilos quando o elemento está sendo ativado, como quando um botão é clicado.

<button>Pressione-me</button>
button:active {
    background-color: lightgray;
}

5. Seletor de Filho

Sintaxe: pai > filho

Descrição: Seleciona elementos que são filhos diretos de um elemento pai.

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ul>
ul > li {
    color: green;
}

  • Item 2
  • Item 3

Resumo da Precedência:

  • 1° Estilo Inline
  • 2° IDs
  • 3° Classes, Atributos e Pseudo-classes
  • 4° Elementos e Pseudo-elementos

Detalhes Adicionais:

Especificidade: A especificidade é um fator importante na precedência. Por exemplo, #id tem uma especificidade mais alta do que .classe, e .classe tem uma especificidade mais alta do que elemento.

Importância: O uso de !important pode aumentar a prioridade de uma regra, independentemente da sua posição na folha de estilos. No entanto, o uso excessivo de !important é desencorajado, pois pode tornar o CSS difícil de manter.

Exemplo: p { color: red !important; }

Ordem de Aparição: Quando dois seletores têm a mesma especificidade, o que aparece por último na folha de estilos será aplicado.

Entender e aplicar corretamente a ordem de precedência é fundamental para evitar conflitos e garantir que o estilo desejado seja aplicado aos elementos da sua página.
