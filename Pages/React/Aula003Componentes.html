O que é Criar UI com Componentes?

UI (User Interface) refere-se à parte visual de um aplicativo com a qual os usuários interagem. Em React, criar uma UI com componentes significa construir a interface do usuário dividindo-a em partes menores e reutilizáveis, chamadas de componentes.

Componentes em React

Componentes são blocos de construção fundamentais em React. Eles permitem que você construa interfaces de usuário de forma modular e reutilizável. Existem dois tipos principais de componentes em React:

Componentes Funcionais:

São funções JavaScript que retornam JSX (uma sintaxe semelhante a HTML). Eles são usados para criar a estrutura da interface do usuário e podem usar Hooks para gerenciar estado e efeitos colaterais.


        import React from 'react';
        
        function MeuComponente() {
          return (
        <div>
          <h1>Olá, mundo!</h1>
          <p>Este é um componente funcional.</p>
        </div>
          );
        }
        
        export default MeuComponente;

Componentes de Classe:

São classes JavaScript que estendem React.Component. Eles têm métodos de ciclo de vida e podem gerenciar o estado de forma interna.


        import React, { Component } from 'react';
        
        class MeuComponente extends Component {
          render() {
        return (
          <div>
            <h1>Olá, mundo!</h1>
            <p>Este é um componente de classe.</p>
          </div>
        );
          }
        }
        
        export default MeuComponente;

Como Componentes Funcionam

  • Reutilização: Você pode criar um componente uma vez e usá-lo em várias partes do seu aplicativo. Isso promove a reutilização de código e a consistência visual.
  • Composição: Componentes podem ser compostos dentro de outros componentes. Isso significa que você pode construir a UI combinando vários componentes menores para formar um componente maior.
  • Gerenciamento de Estado: Componentes podem ter seu próprio estado (usando useState em componentes funcionais ou this.state em componentes de classe). Isso permite que eles atualizem a UI com base em dados que mudam ao longo do tempo.
  • Props: Props são propriedades que você passa para um componente de fora, permitindo que você configure e personalize o comportamento e a aparência dos componentes.

Exemplo de Criação de UI com Componentes

Vamos criar uma simples interface de lista de tarefas:

Componente de Item da Lista:


        // src/components/ItemLista.js
        import React from 'react';
        
        function ItemLista(props) {
          return <li>{props.item}</li>;
        }
        
        export default ItemLista;

Componente de Lista de Tarefas:


        // src/components/ListaDeTarefas.js
        import React from 'react';
        import ItemLista from './ItemLista';
        
        function ListaDeTarefas() {
          const tarefas = ['Estudar React', 'Criar um projeto', 'Ler um livro'];
        
          return (
        <ul>
          {tarefas.map((tarefa, index) => (
            <ItemLista key={index} item={tarefa} />
          ))}
        </ul>
          );
        }
        
        export default ListaDeTarefas;

Componente Principal:


        // src/App.js
        import React from 'react';
        import ListaDeTarefas from './components/ListaDeTarefas';
        
        function App() {
          return (
        <div>
          <h1>Minha Lista de Tarefas</h1>
          <ListaDeTarefas />
        </div>
          );
        }
        
        export default App;

Benefícios de Usar Componentes

  • Modularidade: Divida a UI em partes menores e mais gerenciáveis.
  • Reutilização: Crie componentes reutilizáveis para evitar duplicação de código.
  • Manutenção: Facilita a manutenção e atualização da interface do usuário.

Em resumo, criar uma UI com componentes em React envolve a construção de partes independentes e reutilizáveis da interface, que podem ser combinadas para criar uma aplicação completa e interativa.
