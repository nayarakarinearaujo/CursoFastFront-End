Conceitos Básicos do React

Para construir aplicações com React, é fundamental estar familiarizado com três conceitos principais:

Componentes:

O que são? Componentes são blocos reutilizáveis e independentes que representam uma parte da interface do usuário. Cada componente pode ter seu próprio estado e lógica, e pode ser composto por outros componentes.

Por que são importantes? Eles ajudam a dividir a aplicação em partes menores e mais gerenciáveis, promovendo a reutilização e a modularidade do código.

Estado (State):

O que é? O estado é um objeto que armazena dados que podem mudar ao longo do tempo e afetam a renderização do componente. Cada componente pode ter seu próprio estado, que pode ser atualizado com base em interações do usuário ou outros eventos.

Por que é importante? O gerenciamento do estado permite que você crie interfaces dinâmicas e responsivas, onde a UI se atualiza automaticamente quando os dados do estado mudam.

Propriedades (Props):

O que são? Props são parâmetros passados para um componente de fora, geralmente por seu componente pai. Eles permitem que os componentes recebam dados e funções para personalizar seu comportamento e aparência.

Por que são importantes? As props permitem que você configure e controle a forma como os componentes se comportam e são exibidos, facilitando a comunicação entre componentes e a construção de interfaces complexas a partir de componentes simples.

Exemplos para ilustrar:

Componentes:

// Componente de botão simples
        function Botao(props) {
          return <button>{props.texto}</button>;
        }
        
        // Componente pai que usa o Botao
        function App() {
          return (
        <div>
          <Botao texto="Clique aqui" />
        </div>
          );
        }

Estado (State):

import React, { useState } from 'react';
        
        function Contador() {
          const [contador, setContador] = useState(0);
          
          return (
        <div>
          <p>Contador: {contador}</p>
          <button onClick={() => setContador(contador + 1)}>Incrementar</button>
        </div>
          );
        }

Propriedades (Props):

function Saudacao(props) {
          return <h1>Olá, {props.nome}!</h1>;
        }
        
        function App() {
          return (
        <div>
          <Saudacao nome="João" />
          <Saudacao nome="Maria" />
        </div>
          );
        }

Estar confortável com componentes, estado e props é essencial para criar e gerenciar aplicações React de forma eficaz. Eles são a base para construir interfaces de usuário reativas e bem estruturadas.
