Aula JavaScript - DOM

O que é DOM?

O DOM (Document Object Model) é uma interface de programação que permite aos desenvolvedores interagirem com documentos HTML e XML. Ele representa a estrutura de um documento como uma árvore de nós, onde cada nó corresponde a uma parte do documento, como elementos, atributos, texto, etc.

Como funciona o DOM?

Quando uma página web é carregada, o navegador cria o DOM a partir do HTML da página. Esse modelo permite que linguagens de programação, como JavaScript, acessem e manipulem a estrutura e o conteúdo da página de forma dinâmica.

Componentes do DOM

  • Elementos: Cada tag HTML é representada como um elemento no DOM. Por exemplo, <div>, <p>, e <h1> são todos elementos.
  • Atributos: São propriedades dos elementos, como id, class, src, href, etc.
  • Nós de texto: O conteúdo textual de um elemento, como o texto dentro de um parágrafo <p>, é um nó de texto no DOM.

Manipulação do DOM

JavaScript pode ser usado para alterar a estrutura, o estilo e o conteúdo de uma página após ela ter sido carregada. Isso é feito usando métodos e propriedades do DOM.

Selecionar Elementos

Para manipular elementos, primeiro você precisa selecioná-los.

document.getElementById("id"); // Seleciona um elemento pelo seu ID.
document.getElementsByClassName("classe"); // Seleciona todos os elementos com uma determinada classe.
document.querySelector("seletor"); // Seleciona o primeiro elemento que corresponde a um seletor CSS.

Alterar Conteúdo

Você pode alterar o conteúdo de um elemento usando innerHTML ou textContent.

document.getElementById("exemplo").innerHTML = "Novo conteúdo";

Alterar Estilos

Você pode alterar os estilos CSS de um elemento usando a propriedade style.

document.getElementById("exemplo").style.color = "blue";

Adicionar e Remover Elementos

Você pode criar novos elementos, adicioná-los ao DOM e também remover elementos existentes.

var novoElemento = document.createElement("div");
    novoElemento.innerHTML = "Eu sou um novo elemento!";
    document.body.appendChild(novoElemento);
    
    var elemento = document.getElementById("exemplo");
    elemento.remove();

Exemplo Prático

Aqui está um exemplo simples de como o DOM pode ser manipulado:

<!DOCTYPE html>
    <html lang="pt-BR">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Exemplo de Manipulação do DOM</title>
    </head>
    <body>
        <h1 id="titulo">Olá, Mundo!</h1>
        <button onclick="alterarTitulo()">Clique para alterar o título</button>
    
        <script>
            function alterarTitulo() {
                document.getElementById("titulo").textContent = "Título Alterado!";
            }
        </script>
    </body>
    </html>

Nesse exemplo, ao clicar no botão, o texto do título <h1> é alterado para "Título Alterado!". Isso mostra como o DOM permite interatividade em uma página web.
