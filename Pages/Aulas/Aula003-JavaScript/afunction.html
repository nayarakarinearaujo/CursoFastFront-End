Aula JavaScript - Function

O que é uma função arrow?

As funções arrow são uma sintaxe mais curta para definir funções em JavaScript, introduzida no ES6 (ECMAScript 2015). Elas têm uma estrutura diferente das funções tradicionais e oferecem algumas vantagens, como:

  • Sintaxe mais concisa: Menos código é necessário para definir uma função.
  • Contexto this: As funções arrow não possuem seu próprio this. Em vez disso, elas herdam o this do escopo em que foram definidas, o que pode evitar problemas comuns com o contexto this em funções tradicionais.

Exemplo de função arrow:

const square = (num) => num * num;

Detalhes da função arrow:

  • (num) =>: Esta é a definição da função arrow. (num) é o parâmetro, e => indica que é uma função arrow.
  • num * num: Esta é a expressão que será executada e o valor será retornado automaticamente.

Comparação com funções tradicionais:

Aqui está o mesmo exemplo usando uma função tradicional:

function square(num) {
        return num * num;
        }

Ambas as formas funcionam da mesma maneira, mas a função arrow é mais concisa e evita a necessidade de escrever a palavra-chave function e return (em funções de uma única linha).

Resumindo:

Sim, a forma const square = (num) => num * num; é uma função arrow em JavaScript, caracterizada por sua sintaxe curta e pelo comportamento específico em relação ao this.
